### count VS counta
The COUNT function only counts cells that contain numbers, dates, or times, while COUNTA (Count All) counts every cell that is not empty,     
including numbers, text, logical values (TRUE/FALSE), and errors. Both functions ignore completely blank cells.

### Index match VS Vlookup
"INDEX-MATCH is more flexible and efficient than VLOOKUP because it supports dynamic column referencing, and is less affected    
when columns are inserted or deleted.    

Index / match is for larger dataset     
=INDEX(return_range, MATCH(lookup_value, lookup_range, 0))     
=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])
