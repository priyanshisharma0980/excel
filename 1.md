### count VS counta
The COUNT function only counts cells that contain numbers, dates, or times, while COUNTA (Count All) counts every cell that is not empty,     
including numbers, text, logical values (TRUE/FALSE), and errors. Both functions ignore completely blank cells.

 Xlookup VS Vlookup


 VLOOKUP can only search right. XLOOKUP can search in any direction (up, down, left, right).    
 VLOOKUP defaults to approximate match (dangerous for exact data). XLOOKUP defaults to exact match (safer).   
 If VLOOKUP fails, it returns an #N/A error. XLOOKUP has built-in error messaging using the [if_not_found] argument.

### Wildcards 

(*): Matches any number of characters. (e.g., *east finds "Northeast" and "Southeast").      
(?): Matches exactly one single character. (e.g., sm?th finds "smith" and "smyth").      
(~): Escapes a wildcard so Excel searches for the actual *, ?, or ~ instead of treating it as a wildcard.
